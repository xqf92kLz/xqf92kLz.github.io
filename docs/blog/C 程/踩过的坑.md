??? example "第二次小测前错题"
	while语句的一般形式如下，其中的循环体语句只能是一条语句。T or F

	```cpp
	while (表达式)
		循环体语句
	```
	
	??? note "解答"
		答案：T（不是 F）。
		
		没有大括号。
	
	数组定义后，数组名的值是一个地址，可以被修改。 T or F
	
	??? note "解答"
		答案：F（不是 T）。
		
		C 语言中数组名的本质是 “地址常量”。数组定义后，其在内存中的存储位置是固定的。
	
	数组定义后，只能引用单个的数组元素，而不能一次引用整个数组。T or F
	
	??? note "解答"
		答案：T（不是 F）。
		
		函数参数 `int a[]` 不是整体引用数组，它就是 `int *a`，是在传递一个指针。

??? example "期末考前错题"
    以下程序段的功能是输出 1~100 之间每个整数的各位数字之和。T or F

	```c
	for(num = 1; num <= 100; num++){ 
	    s = 0;
	    do{
	       s = s + num % 10;
	       num = num / 10;
	    }while(num != 0);
	    printf("%d\n", s); 
	}
	```
	
	??? note "解答"
		答案：F（不是 T）
		
		循环体直接把 num 改了


	文件指针指向文件缓冲区中文件数据的存取位置。  T or F
	
	??? note "解答"
		答案：F（不是 T）
		
		文件指针式特殊指针，指向的是文件类型结构，它是多项信息的综合题。每一个文件都有自己的 FILE 结构和文件缓冲区，FILE 结构中有一个 curp 成员，通过 fp->curp 可以只是文件缓冲区中数据存取的位置。
	
	若有定义 `double *p;`，使 p 指向一个双精度型的动态存储单元的语句为（）。
	
	A. `p = malloc(sizeof(double));`
	
	B. `p = (double *)malloc(sizeof(*p));`
	
	C. `p = malloc(8);`
	
	D. `p = (double *)malloc(sizeof(double));`
	
	??? note "解答"
		答案：BD
		
		感觉像有争议的题目
		
		A. 缺少强制类型转换，在 C 语言中虽合法但可能产生警告；
		
		B. `*p` 是 double 类型，因此 `sizeof(*p)` 等价于 `sizeof(double)`
		
		C. 某些特殊平台上 double 不是 8 字节（比如早期系统是 4 字节），会导致分配的空间大小错误，丧失可移植性
