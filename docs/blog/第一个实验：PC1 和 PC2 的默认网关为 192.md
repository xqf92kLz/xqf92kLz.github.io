   第二个实验：PC1 能，PC2 不能；能 ping 通 PC1，不能 ping 通 PC2

3. 外网访问需经过路由器 WAN 口 IP 及端口转发，内网访问直接使用内网 IP

4. 不能

   外网主机的请求数据包可以到达路由器 WAN 口。路由器根据虚拟服务器或 DMZ 规则，将数据包正确转发给内网主机。内网主机处理请求后，需要发送回复数据包给外网主机，但由于内网主机未设置默认网关，它无法将目的地为其他网段的数据包送达至路由器，导致回复中断

5. 尝试访问 PC1；不能访问成功

   当一个 WAN 外网来的数据包到达路由器的 WAN 口时，如果它没有被设置为重定向到 LAN 内网中的任一虚拟服务器，而 DMZ 被启用了，那么该数据包将会被重定向到 DMZ 主机

   PC1 只配置了 Web 服务，没有配置 FTP 服务

6. 虚拟服务器：将内部特定服务暴露给外网，如远程管理家庭智能家居设备

   DMZ：临时让外网完全访问内网某一台设备上的所有服务，如在家中临时搭建一个供多人测试的游戏联机服务器
