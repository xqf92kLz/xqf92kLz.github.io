    实验二 无线路由器、无线AP配置及WLAN共享上网实验 详细步骤与核心要点

    • 实验基础信息

      • 实验分组与设备

        • 分组规则：1 台电脑为 1 小组，3 个相邻小组为 1 大组（共 14 个大组），每大组领取无线路由器、无线 AP 各 1 台（含电源）、直通线 2 根。

        • 核心设备及参数：

          设备型号 核心参数
          TL-XDR1860 无线路由器 双频段（2.4GHz：574Mbps；5GHz：1201Mbps），4 个 WAN/LAN 复用口（盲插），默认 LAN 口 IP：192.168.1.1
          TL-CPE210 无线 AP 单频段（2.4GHz：300Mbps），PoE 网线供电，默认 LAN 口 IP：192.168.1.254，IP55 防尘防水
          TL-XDN8180 无线网卡 双频段（2.4GHz：574Mbps；5GHz：2402Mbps），已预装在实验室电脑

      • 设备关键操作（前置必备）

        • 端口说明：
          • 路由器：4 个网口均支持 WAN/LAN 自适应，无需区分，LAN 口与 WAN 口 IP 必属不同网段。
          • AP：通过黑色 PoE 适配器连接，POE 口供电（连 AP 的 LAN0 口），LAN 口传数据（连电脑 / 交换机）。
        • 指示灯判断：
          • 路由器：顶部系统灯绿色常亮 = 配置完成；红色常亮 = 待配置 / 线路异常；网口灯闪烁 = 正在收发数据。
          • AP：左侧 LAN0 灯亮 = PoE 连接正常；4 个信号灯全亮 = AP 模式（默认），不全亮 = 需复位。
        • 复位操作（忘记 IP 时）：
          • 路由器：通电状态长按 RESET 键，直至顶部系统灯闪烁后松开，重启后恢复默认 IP 192.168.1.1。
          • AP：通电状态长按 PoE 适配器的 Remote Reset 键，直至信号灯全亮全灭后松开，恢复默认 IP 192.168.1.254。

    • 第一部分：无线路由器配置及共享上网实验

      • 实验拓扑校网交换机 ↔ 路由器 WAN 口（任意网口）↔ 路由器 WLAN 口（5GHz）↔ PC1/PC2/PC3（无线连接，禁用有线网卡）

      • 核心参数预设

        • 内网段：192.168.1.*（路由器 LAN 口 IP：192.168.1.1），子网掩码均为 255.255.255.0。

        • PC 静态 IP：PC1=192.168.1.3、PC2=192.168.1.4、PC3=192.168.1.5，默认网关 = 192.168.1.1。

        • DNS 服务器：浙大校网 DNS=10.10.0.21（首选）、10.10.2.21（备用）。

        • 5GHz 信道分配表（避免干扰）：

          大组号 信道号 连接交换机位置 大组号 信道号 连接交换机位置
          一 / 十二 36 前面 / 后面 八 / 九 153/157 前面
          二 / 十三 40 前面 / 后面 十 / 十一 161/165 中间
          三 / 十四 44 中间 / 后面 五 149 中间
          四 48 中间 六 / 七 153/157 后面

      • 详细步骤

        • 阶段 1：有线配置路由器（推荐，更稳定）
          1. 电脑准备：
            • 禁用无线网卡：桌面右下角网络图标→网络和 Internet 设置→更改适配器选项→右键 “WLAN”→禁用。
            • 启用有线网卡：右键 “以太网”→启用，双击 “Internet 协议版本 4（TCP/IPv4）”→手动设置：IP=192.168.1.2，子网掩码 = 255.255.255.0，默认网关 = 192.168.1.1→确定。
          2. 连接路由器：用直通线连接电脑有线网卡与路由器任意网口，确保路由器网口灯闪烁。
          3. 登录配置页面：
            • 打开 Chrome/Firefox 浏览器，地址栏输入 192.168.1.1→首次登录设置管理员密码 = admin123→跳过向导。
          4. 配置 WAN 口（上网设置）：
            • 路由设置→上网设置→WAN 口连接类型 = 自动获得 IP 地址→保存。
            • 确认 “WAN 口设定 = 自动选择 WAN 网口”→保存（确保网口盲插功能生效）。
          5. 配置 LAN 口：
            • 路由设置→LAN 口设置→手动→IP 地址 = 192.168.1.1（保持默认）→子网掩码 = 255.255.255.0→保存。
          6. 配置无线参数（5GHz）：
            • 路由设置→无线设置→关闭 “Wi-Fi 多频合一”→关闭 2.4G 无线功能→打开 5G 无线功能：
              • 无线名称（SSID）：设为 “大组 X-5G”（如 “一组 - 5G”，避免重复）。
              • 无线信道：按上述分配表选择（如大组一选 36）。
              • 认证类型：WPA-PSK/WPA2-PSK→不设密码（实验简化）→保存。
          7. 启用 DHCP 服务器：
            • 路由设置→DHCP 服务器→开启→地址池 = 192.168.1.100-192.168.1.199→地址租期 = 120 分钟→保存。
          8. 重启路由器：路由设置→重启和恢复出厂→重启路由器→等待系统灯绿色常亮。
        • 阶段 2：无线连接测试（本地实验桌）
          1. 电脑禁用有线网卡→启用无线网卡→双击 “TCP/IPv4”→手动设置 IP=192.168.1.3（PC1），默认网关 = 192.168.1.1，DNS=10.10.0.21/10.10.2.21→确定。
          2. 桌面右下角网络图标→找到本组 SSID（如 “一组 - 5G”）→勾选 “自动连接”→连接。
          3. 验证：浏览器输入 192.168.1.1→能打开路由器配置页面 = 无线连接成功。
        • 阶段 3：接入校网并验证
          1. 路由器连接校网：用直通线连接路由器任意网口（自动识别为 WAN 口）与实验室对应位置交换机端口（前三排大组连前面交换机，中间三排连中间，后两排连后面），确保交换机端口灯闪烁。
          2. 电脑远程无线连接路由器→登录配置页面→路由设置→上网设置→确认 WAN 口获得 10.78.18.* 开头的校网 IP（如 10.78.18.8）= 校网连通。
          3. 访问测试：浏览器输入http://speedtest.zju.edu.cn（非 HTTPS）→能测速 = 实验完成。
          4. 其他 PC 操作：PC2/PC3 重复阶段 2，IP 分别设为 192.168.1.⅘，均需能访问校网。
        • 阶段 4：IPv6 配置小实验（选做）
          1. 路由器配置页面→路由设置→IPv6 设置→开启 IPv6 功能：
            • WAN 口连接类型 = 固定 IP 地址→IPv6 地址 = 2025::1，子网前缀长度 = 64，网关 = 2025::2，DNS=2025::3/2025::4→保存。
            • 局域网设置→主机配置 = DHCPv6，IPv6 地址前缀 = 2024::/64→保存。
          2. 电脑无线网卡→TCP/IPv6 属性→自动获取地址。
          3. 验证：Win+R→cmd→输入 ping 2025::1→4 个数据包接收成功 = IPv6 配置生效。

      • 核心要点

        • 必须启用 DHCP 服务器，关闭则路由器网口盲插失效，无法自动识别 WAN/LAN。
        • SSID 必须加 “5G” 字样，信道严格按分配表设置，避免大组间干扰。
        • 电脑不可同时启用有线和无线网卡，否则可能出现 IP 冲突。
        • 若 ping 不通校网，检查 Windows 防火墙是否关闭（控制面板→系统和安全→Windows Defender 防火墙→关闭）。

    • 第二部分：无线 AP 配置及共享上网实验

      • 实验拓扑校网交换机 ↔ AP 的 PoE 适配器 LAN 口 ↔ AP 的 WLAN 口（2.4GHz）↔ PC1/PC2/PC3（无线连接，禁用有线网卡）

      • 核心参数预设

        • 网段：仅 1 个校网网段（PC 自动获取 10.78.18.*），AP 管理 IP=192.168.1.x（x=254 - 大组号，如大组一 = 253）。

        • PC 设置：自动获得 IP 地址、自动获得 DNS 服务器地址。

        • 2.4GHz 信道分配表（避开 ⅙/11 拥挤信道）：

          大组号 信道号 连接交换机位置 大组号 信道号 连接交换机位置
          一 / 十二 2 前面 / 后面 八 / 九 8/9 前面
          二 / 十三 3 前面 / 后面 十 10 中间
          三 / 十四 4 中间 / 后面 十一 12 中间
          四 5 中间 六 / 七 8/9 后面
          五 7 中间

      • 详细步骤

        • 阶段 1：有线配置 AP
          1. 电脑准备：
            • 禁用无线网卡→启用有线网卡→TCP/IPv4 设置：IP=192.168.1.2，子网掩码 = 255.255.255.0，默认网关 = 192.168.1.254→确定。
          2. 连接 AP：用直通线连接电脑与 AP 的 PoE 适配器 LAN 口，AP 的 LAN0 口已预连线到 PoE 适配器 POE 口（供电 + 传数据），确保 AP 的 LAN0 灯亮。
          3. 登录配置页面：
            • 浏览器输入 192.168.1.254→首次登录设置用户名 = admin，密码 = admin123→同意使用条款→跳过快速设置。
          4. 配置无线参数（2.4GHz）：
            • 无线设置→AP 设置：
              • SSID：设为 “大组 X-AP”（如 “一组 - AP”）→开启 SSID 广播。
              • 信道：按分配表选择（如大组一选 2）→无线模式 = B/G/N 混合模式→加密方式 = 无→确定。
          5. 配置 AP 管理 IP：
            • 网络参数→LAN 设置→连接方式 = 静态 IP→IP 地址 = 192.168.1.x（x=254 - 大组号）→子网掩码 = 255.255.255.0→DHCP 服务器 = 禁用→确定。
          6. 保存配置：退出配置页面→重新登录 192.168.1.x→点击 “保存配置”→生效。
        • 阶段 2：无线连接测试（本地实验桌）
          1. 电脑禁用有线网卡→启用无线网卡→TCP/IPv4 设置 = 自动获得 IP 地址→确定。
          2. 连接本组 AP 的 SSID→浏览器输入 192.168.1.x（如 253）→能打开 AP 配置页面 = 连接成功。
        • 阶段 3：接入校网并验证
          1. AP 连接校网：用直通线连接 PoE 适配器 LAN 口与实验室对应交换机端口，确保交换机端口灯闪烁。
          2. 电脑无线连接 AP→等待自动获取 IP（需为 10.78.18.* 开头）。
          3. 访问测试：浏览器输入http://speedtest.zju.edu.cn→能测速 = 实验完成（仅需 1 台 PC 成功即可，避免校网地址耗尽）。

      • 核心要点

        • AP 默认模式为 Access Point（交换机功能），仅转发同一网段数据，不可启用 DHCP（否则与校网 DHCP 冲突）。
        • AP 管理 IP 必须修改为 192.168.1.x（x=254 - 大组号），避免大组间 IP 冲突。
        • 若 AP 信号灯不全亮，需复位恢复为 AP 模式（长按 Remote Reset 键）。
        • 成功连校网后，可能无法访问 AP 配置页面（因 PC 与 AP 不在同一网段，属正常现象）。

    • 补充说明与常见问题解答

      • 关键注意事项
        • 所有设备密码统一设为 admin123，便于管理和后续排查。
        • 网线必须用直通线，连接时确保两端接口灯闪烁（物理连通标志）。
        • 校网访问仅支持http://speedtest.zju.edu.cn，HTTPS 会提示不安全且无法访问。
      • 常见问题查错步骤
        • （1）无法访问路由器 / AP 配置页面
          1. 检查网线是否插紧，网卡是否启用，IP 是否设为 192.168.1.x（x≠1/254）。
          2. 浏览器是否设置代理（关闭代理后重试），换 Chrome/Firefox 浏览器。
          3. 若仍无法访问，执行复位操作（恢复默认 IP），重启设备后重试。
        • （2）无法连校网（路由器）
          1. 确认路由器 WAN 口已获得 10.78.18.*IP（路由设置→上网设置），未获得则检查交换机连接。
          2. PC DNS 是否设为 10.10.0.21/10.10.2.21，未设置则手动添加。
          3. 关闭 Windows 防火墙和 360 安全卫士（可能阻挡连接）。
        • （3）无法连校网（AP）
          1. 确认 AP 的 DHCP 已禁用，PC 已自动获得 10.78.18.*IP。
          2. 检查 AP 的 LAN0 灯和交换机端口灯是否亮，网线是否插错 PoE 端口。
          3. 若 PC 仅获得 169.254..，禁用无线网卡→启用→重新获取 IP。
      • 选做实验：基于 AP 的 VPN 共享上网
        • 前提：将 AP 配置为 AP Router 模式（支持 L2TP 协议）。
        • 步骤：AP→工作模式→AP Router→上网方式 = L2TP→输入校网 VPN 账号密码→PC 无线连接 AP→共享 VPN 访问校外网。
        • 要点：AP 的 LAN 口变 WAN 口，需有线接入校网，PC 无线接入 AP，AP 管理 IP 末位改为非 254。
